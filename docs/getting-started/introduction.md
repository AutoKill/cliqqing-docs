# Introduction

Welcome to the Cliqqing Discord bot! This guide will help you get started with setting up and using the bot.

## What is Cliqqing?

Cliqqing is a feature-rich Discord bot built with the Sapphire Framework that helps server administrators manage their communities effectively. It provides:

- **Welcome System**: Automatically welcome new members with customizable messages
- **Self-Roles**: Allow members to assign themselves roles through reactions
- **Configuration System**: Flexible configuration with JSON/YAML support
- **Modern Architecture**: Built with TypeScript, Prisma, and modern development tools
